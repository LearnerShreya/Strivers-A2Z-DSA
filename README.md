# Striver's A2Z DSA Notes & Code

## 📌 Overview
This repository contains my notes and code implementations from the **Striver's A2Z DSA** roadmap. The goal is to strengthen my **Data Structures and Algorithms (DSA)** knowledge systematically, covering all important concepts from **basics to advanced topics**.

## 📖 Structure
The repository is structured as per Striver's roadmap, covering topics in a step-by-step manner:

1. **Basics of Programming**
   - Time & Space Complexity
   - Basics of Python/C++
   - Pattern Problems

2. **Basic Data Structures**
   - Arrays, Strings
   - Hashing, Recursion
   - Bit Manipulation

3. **Sorting & Searching**
   - Binary Search
   - Sorting Algorithms

4. **Mathematical Concepts**
   - GCD, LCM, Prime Numbers
   - Modular Arithmetic, Sieve

5. **Recursion & Backtracking**
   - Subsets, Permutations, N-Queens
   - Sudoku Solver, Rat in a Maze

6. **Linked List, Stack & Queue**
   - Single & Doubly Linked List
   - Stack & Queue Implementations

7. **Binary Trees & BST**
   - Traversals, Height, Diameter
   - LCA, Boundary Traversal, BST Operations

8. **Heap, Hashing & Graphs**
   - Min/Max Heap, HashMaps
   - Graph Traversal (BFS, DFS), Shortest Paths

9. **Dynamic Programming**
   - 1D, 2D DP, LIS, LCS
   - DP on Trees, DP on Graphs

10. **Advanced Topics**
    - Tries, Segment Trees, Disjoint Set
    - Fenwick Tree, Bitmasking

## 🛠 Technologies Used
